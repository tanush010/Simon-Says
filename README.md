# 🎮 Simon Says
 
A browser-based implementation of the classic **Simon Says** memory game, built with vanilla HTML, CSS, and JavaScript.
 
---
 
## 📁 Project Structure
 
```
simon-says/
├── simon.html   # Game layout and structure
├── simon.css    # Styling and button flash animation
└── simon.js     # Game logic and event handling
```
 
---
 
## 🚀 How to Run
 
1. Clone or download the project files.
2. Open `simon.html` in any modern web browser.
3. Press **any key** to start the game.
No build tools, dependencies, or server required.
 
---
 
## 🎯 How to Play
 
1. **Start** — Press any keyboard key to begin.
2. **Watch** — The game flashes a sequence of colored buttons.
3. **Repeat** — Click the buttons in the exact same order.
4. **Level Up** — Each correct round adds one more step to the sequence.
5. **Game Over** — A wrong click ends the game and shows your score.
6. **Restart** — Press any key again to play a new round.
Your **high score** is tracked and displayed throughout the session.
 
---
 
## ✨ Features
 
- 4 colored buttons: Red, Yellow, Green, Blue
- Randomized sequences that grow each level
- Visual flash feedback for both game and user presses
- High score tracking (session-based)
- Game over screen with score display and red background flash
- Clean reset and restart flow
